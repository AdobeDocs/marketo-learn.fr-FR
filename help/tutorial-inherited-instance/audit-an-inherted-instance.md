# Comment auditer une instance Marketo Engage héritée

Hériter d’une instance de Marketo Engage peut être une période stressante, que vous soyez un administrateur expérimenté ou un administrateur plus récent, il peut être difficile de savoir où commencer le nettoyage. Pour commencer, nous vous recommandons de prendre le temps de comprendre ce qui se trouve dans l’instance dont vous venez d’hériter aujourd’hui et de créer un plan d’action pour mettre à jour la configuration.

Dans ce tutoriel, nous vous guiderons à travers les sections les plus importantes de Marketo Engage, notamment Admin, Activités marketing, Design Studio et Base de données, et vous donnerons des conseils sur l’audit et l’optimisation tout au long du processus.
